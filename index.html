Comandos Basicos

Query

Uma query em SQL é uma instrução escrita na linguagem SQL (Structured Query Language) que é usada para recuperar, inserir, atualizar ou excluir dados em um banco de dados. As queries são usadas para comunicar-se com o banco de dados e executar operações desejadas.

Comando de consulta

Soleciona a tabela selecionada

Select * from TB_VENDEDORES
Incersão de dados

Incerre dados na tabela selecionada definindo a tabela e os campos, logo em seguida
definindo os valores com os mesmos respectivos lugares abaixo do VALUE

INSERT INTO TB_VENDEDORES
(MATRICULA, NOME, DATA_ADMISSAO, PERCENTUAL_COMISSAO)
VALUES
('00810','Marcia Almeida','14/12/2016', 0.18);
Deletar tabelas

Deleta a tabela

DROP TABLE TB_VENDEDORES
Ataluzação de tabela

Atualiza a tabela selecionada
UPDATE seleciona a tabela desejada
SET escolhe qual campo e valor da tebela vai ser trocado
WHERE faz o filtro de qua valor definido deve ser trocado
Colocando mais uma Virgula no SET pode adicionar mais vaores para fazer a alteração

UPDATE TB_VENDEDORES
SET PERCENTUAL_COMISSAO = 0.18
WHERE MATRICULA = '00400';
Deleta um linha de uma tabela

Deleta a linha de uma tabela selecionada
DELETE FROM faz a selecão da tabela onde vai ser feita a alteração
WHERE faz a seleção de qual campo a alteração vai ser feita

DELETE FROM TB_VENDEDORES WHERE MATRICULA = '00400';
Alterando valor e restrição da tabela

ALTER TABLE faz a seleção da tabela e já avisa que vai ser feita alguma alteração
ADD CONSTRAINT vai define o nome da nova tabela que vai ser cirado
PRIMARY KEY define que essa coluna vai ser um chave primeria e em parentes define a coluna que vai ser atualizada
PRIMARY KEY São restrições de banco que difine o id da coluna como pai
e não podendo dexar ele com valor nulo ou igual a outro já existente

ALTER TABLE TB_VENDEDORES
ADD CONSTRAINT PK_TB_VENDEDORES
PRIMARY KEY (MATRICULA);
Especificação do metodo TO_DATE para leitura de Data

TO_DATE nada mais é que uma função que recebe dois valores formatando para o formato DATE.

INSERT INTO TB_VENDEDORES ( MATRICULA, NOME, DATA_ADMISSAO, PERCENTUAL_COMISSAO ) VALUES ( '00777','Katy Peterson',TO_DATE('02/04/2020','MM/DD/YYYY'), 0.10 );
Comando de consulda de data especifica

A função TO_CHAR traz os indices das especificação definida do comando acima por exemplo o indice MM
em poucas palavras é como de tivese definindo uma variavel em uma lista de array

SELECT TO_CHAR(DATA_NACIMENTO,MM) FROM TB_CLIENTES
Operadores Logicos

Operadores Logicos tbm tem a mesma função em SQL caso a consulta tenha alguma especialização de chamada

SELECT * FROM TB_VENDEDORES WHERE PERCENTUAL_COMISSAO < 0.17;
SELECT * FROM TB_VENDEDORES WHERE PERCENTUAL_COMISSAO <= 0.17;
SELECT * FROM TB_VENDEDORES WHERE PERCENTUAL_COMISSAO => 0.17;
SELECT * FROM TB_VENDEDORES WHERE PERCENTUAL_COMISSAO <> 0.17;
Operadores Logicos AND OR

AND e OR tem a mesma logica que na programação eles validão as logicas definidas
retornan o valor delas se for verdadeira

SELECT * FROM TB_CLIENTES WHERE (IDADE >= 18 AND SEXO = 'F' AND CIDADE = 'Sao Paulo') OR BAIRRO = 'Barra da Tijuca';
Operadores Logicos NOT

Inverto o valor logico Verdadeiro ou falso

SELECT * FROM TB_CLIENTES WHERE NOT((IDADE >= 18 AND SEXO = 'F' AND CIDADE = 'Sao Paulo') OR BAIRRO = 'Barra da Tijuca');
Ateração de Identificador

Coverte o nome da tabela.

SELECT CPF AS IDENTIFICADOR, NOME AS "NOME DE CLIENTE", BAIRRO, CIDADE FROM TABELA_DE_CLIENTES;
Ateração de Identificador de tabela

Um simples espaço depois de chamar a tabela voce cria uma espece gancho da tabela.

SELECT TDC.* FROM TABELA_DE_CLIENTES TDC;
Between

Between pega o espaço entre duas operações "ENTRE"

SELECT * FROM TABELA_DE_CLIENTES WHERE IDADE BETWEEN 17 AND 22;
Chamando varios Valores

O IN serve para quando precisar de varios valores ou strings
de valores diferente em ver de usar o OR pede-se usar IN

SELECT * FROM TABELA_DE_PRODUTOS WHERE SABOR IN ('Manga','Laranja','Melancia') AND TAMANHO = '1 Litro';
Busca de caracteres em Strings

Em uma busca de caracteres em uma descrição a função LIKE pega a string entre aspas e busca na coluna
observação: as % seja na frente como no final da string descreve os parametro da string definida
como a string do banco de dados deve estar.

SELECT * FROM TABELA_DE_PRODUTOS WHERE SABOR LIKE '%Maca%';
